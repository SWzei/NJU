# NJU林泉钢琴社 网站社员使用手册

本文档面向普通社员，用于说明注册、登录、琴时志愿填写、音乐会报名、资料维护、目录查看和通知接收的完整使用流程。

**首次使用**可通过顶部导航 `注册` 进入注册页，填写学号、邮箱、姓名和密码后提交；其中学号与密码是关键字段，姓名建议使用真实姓名以便社内识别。注册后可通过顶部导航 `登录` 进入，支持“学号 + 密码”或“邮箱 + 密码”两种方式；登录后系统会显示与社员身份对应的功能入口，界面默认中文，也可在右上角切换中英文。

**首页**主要用于浏览信息，包括社团活动、公告通知和影集展示内容；当管理员发布新活动或公告后，首页会优先显示最新信息。

**琴时安排**入口在顶部导航 `琴时安排`；你可以在周历里选择任意数量时段，每个时段会显示当前已有多少人选择，点击格子即可选中或取消，最后点击保存提交本学期琴时志愿。排班发布后在同一页面查看个人结果，系统目标是每位成员每周 1-2 小时琴时。
（**注意**：报名琴课的社员需预先报名相应琴时，予以优先安排）

**音乐会**功能入口在顶部导航 `钢琴乐会`；报名时需要按页面字段填写姓名、学号、曲目作者与名称（中/英）、预计时长、QQ 联系方式，并可附加备注和乐谱文件（PDF/图片）。支持你查看并修改已提交报名内容；如果你已经提交过，页面会展示“我的已提交报名信息”，你可以在原表单更新后再次提交，管理员端会读取你最新一次提交结果。

**审核阶段**你可在音乐会页面查看审核时段、结果状态与管理员反馈；若暂未安排或暂未出结果，页面会显示相应提示信息。

**个人资料**入口为 `我的资料`，你可维护姓名、头像、个人照片、简介、年级、院系、专业、兴趣爱好、钢琴兴趣、微信号和电话等字段；头像与个人照片支持本地上传，保存后会立即生效。

**社员目录**入口为 `林泉相册`；你可以按姓名、学号、院系或专业搜索成员，并点击卡片进入详情页查看对方公开信息。

**常见问题**

* 系统通知目前以管理员触发为主，常见场景包括排班发布、音乐会发布或结果更新、重要公告；若未收到邮件，通常是因为账号未填写有效邮箱，建议先在个人资料中补全。

* 登录失败通常由账号或密码输入错误引起，建议先核对后重试；志愿保存失败建议刷新后重试并检查网络；乐谱文件若显示随机文件名但可正常下载打开，通常是服务器命名策略，不影响实际使用；看不到新公告或排班结果时，多数情况是管理员尚未正式发布。

* 为了提高使用效率，建议你在注册时填写可用邮箱、在琴时页面**尽量多选**可接受时段、音乐会报名尽量一次填完整、资料页至少维护姓名与联系方式，便于社内协作与通知触达。
